# RL LAB — 強化学習ロボット体験

オープンキャンパス向けの、iPad対応・依存ライブラリなしのWebシミュレータです。

## 起動

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。シミュレータはオフラインで動作します。実機接続にはWeb Bluetooth対応ブラウザとHTTPS配信が必要です。

## 体験フロー

1. 報酬パラメータを調整してAIの性格を設計
2. 15秒間、24体を並列評価する実学習をリアルタイム表示
3. 学習済みモデルをtoioへ転送するSim-to-Realデモ
## 実学習モデル

依存ライブラリなしの進化戦略で `5入力 → 8中間 → 2モーター出力` のニューラルネットワークを学習します。24体を並列シミュレーションし、ゴールへの接近・到達、障害物への接近・衝突、移動速度から報酬を計算します。画面の試行回数、成功率、報酬、ネットワーク重みはすべて実際の評価結果です。学習済みの同じ重みをtoioの左右モーター推論にも使用します。
