# RL LAB — 強化学習ロボット体験

オープンキャンパス向けの、iPad対応・依存ライブラリなしのWebシミュレータです。

## 起動

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。シミュレータはオフラインで動作します。実機接続にはWeb Bluetooth対応ブラウザとHTTPS配信が必要です。

## 体験フロー

1. 報酬パラメータを調整してAIの性格を設計
2. 15秒間・3万試行の学習をリアルタイム表示
3. 学習済みモデルをtoioへ転送するSim-to-Realデモ
