# 固定実機コース R01 — A3配置図

- PDF: `real-course-a3.pdf`
- 編集・ブラウザー印刷用: `real-course-a3.svg`
- PDF生成元: `real-course-a3.ps`

A3横、倍率100%（実際のサイズ）で印刷してください。「用紙に合わせる」などの自動拡大縮小は無効にし、印刷後にコース外枠が `380 × 257 mm`（各辺から20 mm）であることを確認します。

図はA3用紙の四辺から20 mmだけ余白を取り、残りを実機コースとして使用します。障害物は `O1`〜`O3` の円へ置き、toioは `START` の中心へ約−22度（右上向き）で置きます。紫色の円がゴール判定範囲です。

| 対象 | A3左上からの中心位置 (mm) | 印刷上の判定範囲 |
|---|---:|---:|
| START | (74.9, 200.8) | toio本体 32 × 32 mmの目安 |
| O1 | (134.0, 81.8) | 直径28.4 mmの円 |
| O2 | (212.1, 193.0) | 直径33.0 mmの円 |
| O3 | (294.4, 104.0) | 直径31.2 mmの円 |
| GOAL | (368.3, 64.5) | 直径41.2 mmの円 |

印刷図そのものにはPosition IDパターンがないため、toioを走らせる面としては使えません。実際のマット上へ障害物を配置するための原寸ガイドとして使用してください。

Position IDの座標値は仕様上ミリメートル単位ではありません。アプリでは各シートの生座標から約15ずつ内側を実機コース端とし、A3上で約20 mmの物理余白になるよう正規化しています。障害物の中心配置は学習環境E05/R01の値をそのまま拡大したものです。

## 会場装飾用バックボード

- 印刷用PDF: `reinforcement-learning-a0-backboard.pdf`
- 内容確認用PNG: `reinforcement-learning-a0-backboard.png`
- 生成スクリプト: `generate_a0_backboard.py`
- 使用フォント: `fonts/NotoSansJP-VF.otf`（SIL Open Font License 1.1）

仕上がりは `660 × 537 mm` です。A0対応プリンターへこの原寸を指定し、倍率100%（実際のサイズ）で印刷してください。中央の白い `420 × 297 mm` の位置へA3のtoio走行シートを重ねます。背景ポスターの印刷面が走行面へ露出せず、位置合わせの枠と周囲の装飾だけが見えるようにしてください。

A3走行シートの上下左右それぞれ12 cmだけを使用し、指定タイトル、AI・ロボットを連想する装飾、体験の目的を配置しています。走行面には印刷せず、スタート位置を左・下の水色矢印、ゴール位置を上・右の黄色矢印の交点で示します。PDFは原寸・150 dpiです。再生成は `python3 print/generate_a0_backboard.py` で行います。
